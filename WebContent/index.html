Network Statistics SDN Application runs on-top of a OpenDaylight SDN Controller.
It tracks port, table and flow statistics of all the nodes added to OpenDaylight. Run this application on tomcat locally and supply the credentials below to see it in action.

This is a pure jQuery/Bootstrap and HTML based application without any backend in itself.

Table Statistics
